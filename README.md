# AI Impact on Students

This Data Science project analyzes the impact of Generative AI on students' academic performance, skill retention, and burnout risk. The project includes data cleaning, exploratory data analysis (EDA), data visualization, and predictive modeling using Machine Learning.

## Dataset
- 50,000 student records
- 16 features
- No missing values

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model
Random Forest Classifier with Hyperparameter Tuning

**Best Accuracy:** 53.37%

## Objective
To explore how AI usage affects student learning outcomes and predict burnout risk levels using machine learning techniques.
